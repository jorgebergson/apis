    Códigos de Retorno

    Las solicitudes a la API de NFE pueden devolver el siguiente código HTTP:

    Códigos no emitidos

    Los códigos 400 (solicitud no válida) y 500 (error interno del servidor) no se emitirán en la facturación.

    Código / Respuestas

    200 OK

    Todo funcionó como se esperaba y los datos se validaron con éxito.

    206 Contenido parcial

    La información fue devuelta, pero no completamente.

    400 Solicitud inválida

    El número de NFE ingresado no es válido.

    401 No autorizado

    Problemas durante la autenticación. Este error ocurre cuando hay una ruta incorrecta en la solicitud. Asegúrese de llamar a la API de la siguiente manera: "https://apigateway.serpro.gov.br/consulta-nfe-df/v1/nfe/{ni}"

    404 no encontrado

    No hay NFE con el número de registro informado.

    500 Error del servidor

    Se ha producido un error interno del servidor.

    504 Tiempo de espera de puerta de enlace

    Hubo un error de red y la puerta de enlace no respondió a tiempo. La solicitud no llegó al API Consulta NFE.
